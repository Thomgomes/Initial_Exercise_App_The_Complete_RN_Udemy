## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Thomgomes/Initial_Exercise_App_The_Complete_RN_Udemy.git
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npx expo start

  ou

  expo start
  
  ou 

  npm start
```
